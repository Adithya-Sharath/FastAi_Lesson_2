# Car Damage Classifier

This is an example site created for the purpose of studying GitHub Pages deployment and basic fastai model creation and usage. The classification used is "Damaged car and Non-Damaged car".

## Live Demo

Try the model here: [Car Damage Classifier on Hugging Face](https://huggingface.co/spaces/adugudu/FastaiLesson-2)

## About

Built using fast.ai and deployed on Hugging Face Spaces. The model was trained on images of damaged and undamaged cars using transfer learning with ResNet18.
